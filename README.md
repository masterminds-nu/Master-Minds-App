# Masterminds App

Mobil PWA för Masterminds.nu — affärsnätverket för kvinnliga entreprenörer.

Byggd med React + Vite + Supabase.

## Funktioner
- Hem med personlig översikt
- Uppdragstavlan (assignments)
- Nätverk (medlemskatalog)
- CRM (privata kontakter)
- Digitalt visitkort
- Impact dashboard
- AI-rådgivare (Fråga Masterminds)
- Investerarmatchning
- Onboarding

## Utveckling
```
npm install
npm run dev
```

## Build
```
npm run build
```
Output hamnar i `dist/`.
